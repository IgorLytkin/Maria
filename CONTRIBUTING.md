# Внесение вклада в Maria

Спасибо за интерес к нашему проекту! Мы рады принимать вклад от сообщества.

## Как помочь проекту

### Отправка Issues

Если вы нашли ошибку или имеете идею для улучшения:

1. Проверьте, не существует ли уже такой issue
2. Создайте новый issue с подробным описанием
3. Приложите примеры кода или скриншоты, если возможно

### Отправка Pull Requests

1. **Fork** репозиторий
2. Создайте **feature branch** (`git checkout -b feature/amazing-feature`)
3. Сделайте ваши изменения
4. Закоммитьте изменения (`git commit -m 'Add amazing feature'`)
5. Push в ветку (`git push origin feature/amazing-feature`)
6. Откройте **Pull Request**

## Рекомендации по коду

- Используйте понятные названия переменных
- Добавляйте комментарии к сложным операциям
- Тестируйте изменения перед отправкой PR
- Следуйте стилю кода проекта

## Вопросы?

Создайте issue с меткой `question` или напишите в discussions.

Спасибо за вклад! 🎉
